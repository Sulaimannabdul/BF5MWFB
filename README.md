# SocialBox
SocialBox is a Bruteforce Attack Framework for social media  
# Installation
```
apt-get update
apt-get install git
git clone https://github.com/samsesh/SocialBox-Termux.git 
cd SocialBox-Termux
chmod +x install-sb.sh
./install-sb.sh
```
# Run
```
./SocialBox.sh
```
