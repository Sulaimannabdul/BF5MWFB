## [Brute_Force](https://github.com/pxcs/bruframe/) Framework
SocialBox is a Bruteforce Attack Framework for social media  
### Installation
```
apt-get update
apt-get install git
git clone https://github.com/pxcs/bruframe.git 
cd BF5MWFB
chmod +x install-sb.sh
./install-sb.sh
```
### Run
```
./SocialBox.sh
```
